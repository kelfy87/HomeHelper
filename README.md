## 📌 О проекте

**[Home Helper](https://kelfy87.github.io/HomeHelper/)** — это сайт, который помогает быстро находить нужные учебные материалы без лишнего поиска, рекламы и переходов по множеству страниц.

Сервис позволяет за несколько секунд собрать готовую ссылку на нужный предмет, страницу, параграф или номер задания.  
Интерфейс выполнен в приятных кремово-оранжевых тонах, а простая структура делает использование сайта понятным и удобным.

---

## ✨ Возможности

- 📚 Выбор нужного предмета
- 🇬🇧 Для английского языка — переключение между **учебником** и **рабочей тетрадью**
- 🔢 Удобный ввод страницы, параграфа или номера задания

- ↩️ Быстрый переход по ранее созданным ссылкам
- 🧩 Создание пользовательских шаблонов для дальнейшего использования

---

## 🎯 Зачем нужен Home Helper?

Обычно для поиска нужного учебного материала приходится:

- открывать несколько сайтов;
- искать нужный предмет;
- выбирать класс, страницу или номер;
- закрывать рекламу;
- проверять, подходит ли найденная страница.

**Home Helper** сокращает этот процесс до нескольких действий.  
Вы выбираете параметры, вводите нужные данные и сразу получаете готовую ссылку.

---

## 🚀 Как пользоваться

1. Откройте сайт: **[Home Helper](https://kelfy87.github.io/HomeHelper/)**
2. Выберите нужный предмет
3. Введите страницу, параграф или номер задания
4. Соберите ссылку
5. Скопируйте её или перейдите по ней
