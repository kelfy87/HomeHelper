### Для чего нужен?
[Home Helper](https://kelfy87.github.io/HomeHelper/) - это быстрое решение повседневных задач. Благодаря сайту вам больше не придется тратить время на поиск паралели, нужного предмета, подходящего учебника, параллельно просматривая большóе количество рекламы. С помощью сайта можно будет получить желаемые ответы или решения в течение 38 секунд. Оформление сайта в кремово-оранжевых тонах радует глаз, а интуитивно понятный интерфейс делает процесс получения ответов максимально комфортным.
